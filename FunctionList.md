# MP-BOX 功能清單

## 功能列表

| 功能 | 說明 |
|------|------|
| 登入 / 登出 | 使用 email 登入系統，取得 JWT token |
| 首頁 Dashboard | 系統整體狀態總覽 |
| AI 夥伴選擇 | 以卡片列表選擇要使用的 AI 夥伴 |
| 安全事件清單 | 列出所有安全事件，支援篩選與排序 |
| 安全事件詳情 | 查看單一安全事件的完整資訊（左側 tabs） |
| 右側諮詢聊天面板 | 針對安全事件與 AI 夥伴即時對話諮詢 |
| 知識庫列表 / 詳情 | 管理與查閱知識庫內容 |
| 帳號管理 + 角色管理 | 新增 / 編輯 / 停用帳號，管理角色與權限 |
| AI 夥伴管理設定 | 設定與管理 AI 夥伴的參數與配置 |
