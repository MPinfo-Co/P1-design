# 規格撰寫規則與功能編號的關聯

1. 規格檔案存放位置: `P1-design/Spec/`
2. 每一支功能對應一個目錄，目錄名稱 = 功能編號，如: `P1-design/Spec/fn_user/`
3. 每一個功能可對應一個以上畫面，每個頁面一隻規格
4. 規格命名規則「功能名稱_畫面序號_畫面簡述」，如: `fn_user_01_list.md`
