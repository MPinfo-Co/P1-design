# 查詢畫面

[▶ 開啟雛形](../../Prototype/fn_role.html#roleList)

## 欄位說明

### 篩選列

| 欄位       | 元件         | 說明         |
| -------- | ---------- | ---------- |
| **關鍵字**  | text input | 搜尋範圍：**角色名稱** |

### 清單表格

| 欄位           | 資料來源                                  | 顯示格式              |
| ------------ | ------------------------------------- | ----------------- |
| **角色名稱**     | tb_roles.name                         | 純文字               |
| **可用 AI 夥伴** | tb_ai_partners.name（透過 tb_role_ai_partners） | 逗號分隔，無則顯示「無權限」 |
| 執行動作         | —                                     | 修改按鈕 / 刪除按鈕       |

## 操作說明

**[新增角色]**（頁面右上角按鈕）
- → 角色表單（新增模式）

**[套用]**（篩選列按鈕）
- → `Api/fn_role_query_api.md`
  - 傳入：**關鍵字**

**[修改]**（各列操作欄按鈕）
- 帶入該列資料 → 角色表單（修改模式）

**[刪除]**（各列操作欄按鈕）
- 顯示確認訊息：「確定要刪除此角色嗎？無法復原。」
- 確認後 → `Api/fn_role_del_api.md`
  - 傳入：**角色名稱**
  - 成功：重新整理清單
  - 失敗：顯示錯誤訊息
