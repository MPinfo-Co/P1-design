# 新增畫面/修改畫面

雛形畫面：[▶ 開啟雛形](../../Prototype/fn_role.html#roleModal)

## 欄位說明

| 欄位        | 輸入元件           | 必填  | 驗證規則         | 預設值  |
| --------- | -------------- | --- | ------------ | ---- |
| **角色名稱**  | text input     | 是   | 不可空白，系統內不可重複 | 空白   |
| **成員**    | checkbox group | 否   | —            | 全不選  |
| **功能權限**  | checkbox group | 否   | —            | 全不選  |

> **成員**選項來源：tb_users 所有使用者，含「全選」切換按鈕。

### 功能權限選項

| 權限項目   | 說明               |
| ------ | ---------------- |
| 使用者管理  | 可新增、修改、刪除使用者     |
| 角色管理   | 可新增、修改、刪除角色與權限   |

## 操作說明

**[儲存]**
- → `Api/fn_role_add_api.md`
  - 傳入：**角色名稱**、**成員**、**功能權限**
  - 成功：關閉表單 → **查詢畫面**，重新整理清單

**[取消]**
- 關閉表單 → **查詢畫面**，不儲存
