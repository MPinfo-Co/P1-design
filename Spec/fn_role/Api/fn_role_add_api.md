# 介面說明

| 項目       | 內容                                     |
| -------- | ---------------------------------------- |
| Method   | POST                                     |
| Endpoint | /api/roles                               |

# 傳入參數

## 參數類型：Request Body

| 欄位             | 型別        | 必填  | 說明            |
| -------------- | --------- | --- | ------------- |
| **角色名稱**       | string    | 是   | 角色顯示名稱        |
| **成員帳號**       | integer[] | 否   | 指派至此角色的使用者    |
| **可用 AI 夥伴**   | integer[] | 否   | 此角色可使用的 AI 夥伴 |
| **可存取知識庫**     | integer[] | 否   | 此角色可存取的知識庫    |
| **功能權限**       | object    | 否   | 見下方結構         |

**功能權限結構**

| 欄位               | 型別      | 說明        |
| ---------------- | ------- | --------- |
| ai_partner       | boolean | AI 夥伴使用權  |
| manage_accounts  | boolean | 帳號管理權     |
| manage_roles     | boolean | 角色管理權     |
| edit_ai          | boolean | AI 夥伴管理權  |

# 處理邏輯

## 檢核

| 檢核項目       | 失敗條件          | 回應訊息              |
| ---------- | ------------- | ----------------- |
| JWT 驗證     | 未登入或 Token 過期 | 401 未登入或 Token 過期 |
| 權限         | 無 `can_manage_roles` | 403 您沒有執行此操作的權限 |
| **角色名稱**   | 未填寫           | 400 角色名稱未填寫       |
| **角色名稱** 唯一性 | 已存在         | 400 此角色名稱已存在      |

## 執行

- 寫入 tb_roles
- 逐筆寫入 tb_user_roles（成員帳號）
- 逐筆寫入 tb_role_ai_partners（可用 AI 夥伴）
- 逐筆寫入 tb_role_kb_map（可存取知識庫）

# 傳回結果

| Response | Message  | data |
| -------- | -------- | ---- |
| 201      | 新增成功     | 無    |
| 4xx      | 依據檢核邏輯   | 無    |
