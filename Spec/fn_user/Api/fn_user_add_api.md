# 介面說明

| 項目       | 內容                                       |
| -------- | ---------------------------------------- |
| Method   | POST                                     |
| Endpoint | /api/users                               |

# 傳入參數

## 參數類型：Request Body

| 欄位        | 型別        | 必填  | 說明         |
| --------- | --------- | --- | ---------- |
| **名稱**    | string    | 是   | 使用者顯示名稱    |
| **Email** | string    | 是   | 登入信箱       |
| **密碼**    | string    | 是   | 登入密碼       |
| **角色**    | integer[] | 是   | 指派角色       |

# 處理邏輯

## 檢核

| 檢核項目        | 失敗條件          | 回應訊息                  |
| ----------- | ------------- | --------------------- |
| JWT 驗證      | 未登入或 Token 過期 | 401 未登入或 Token 過期     |
| 權限          | 無 `can_manage_accounts` | 403 您沒有執行此操作的權限 |
| **名稱** 格式   | 超過 100 字      | 400 使用者顯示名稱不可超過 100 字 |
| **Email** 格式 | 格式不符         | 400 登入信箱格式不正確         |
| **密碼** 格式   | 少於 8 字元       | 400 密碼最少 8 字元         |
| **角色**      | 為空            | 400 角色未設定             |
| **Email** 唯一性 | 已存在         | 400 此 Email 已被使用      |

## 執行

- 以 bcrypt hash **密碼**，寫入 tb_users（`is_active = true`）
- 逐筆寫入 tb_user_roles

# 傳回結果

| Response | Message    | Result Field |
| -------- | ---------- | ------------ |
| 201      | 新增成功       | 無            |
| 4xx      | 依據檢核邏輯     | 無            |
