# 介面說明

| 項目       | 內容                                       |
| -------- | ---------------------------------------- |
| Method   | POST                                     |
| Endpoint | /api/users                               |

# 傳入參數

## 參數類型：Request Body

| 欄位        | 型別        | 必填  | 說明         |
| --------- | --------- | --- | ---------- |
| **名稱**    | string    | 是   | 使用者顯示名稱    |
| **Email** | string    | 是   | 登入信箱       |
| **密碼**    | string    | 是   | 登入密碼       |
| **角色**    | integer[] | 是   | 指派角色       |

# 處理邏輯

1. 驗證 JWT，確認使用者具 `can_manage_accounts = true`
2. 確認 **Email** 在 tb_users 中不重複
3. 以 bcrypt hash **密碼**，寫入 tb_users（`is_active = true`）
4. 逐筆寫入 tb_user_roles

# 傳回結果

| Response | Message              | Result Field |
| -------- | -------------------- | ------------ |
| 201      | 新增成功                 | 無            |
| 400      | 使用者顯示名稱不可超過 100 字    | 無            |
| 400      | 登入信箱格式不正確            | 無            |
| 400      | 密碼最少 8 字元            | 無            |
| 400      | 角色未設定                | 無            |
| 400      | 此 Email 已被使用         | 無            |
| 401      | 未登入或 Token 過期        | 無            |
| 403      | 您沒有執行此操作的權限          | 無            |
