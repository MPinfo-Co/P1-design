# 查詢畫面

[▶ 開啟雛形](../../Prototype/fn_user.html#userList)

## 欄位說明

### 篩選列

| 欄位       | 輸入元件         | 說明                                  |
| -------- | ---------- | ----------------------------------- |
| **角色職位** | select     | 選項來源：tb_roles 所有角色名稱，加上「全部」選項（預設選中） |
| **關鍵字**  | text input | 搜尋範圍：**姓名**、**電子信箱**                |

### 清單表格

| 欄位       | 顯示元件                      | 資料來源                            |
| -------- | --------------------------- | ------------------------------- |
| **姓名**   | 純文字                         | tb_users.name                   |
| **電子信箱** | 純文字                         | tb_users.email                  |
| **角色**   | badge（可多個）                  | tb_roles.name（透過 tb_user_roles） |


## 操作說明

**[新增帳號]**（頁面右上角按鈕）
- → 新增畫面

**[套用]**（篩選列按鈕）
- → `Api/fn_user_query_api.md`
  - 傳入：**角色職位**、**關鍵字**

**[修改]**（各列操作欄按鈕）
- 帶入 **電子信箱** → 修改畫面

**[刪除]**（各列操作欄按鈕）
- 顯示確認訊息：「確定要刪除此帳號嗎？無法復原。」
- 確認後 → `Api/fn_user_del_api.md`
  - 傳入：**電子信箱**
  - 成功：重新整理清單
  - 失敗：顯示錯誤訊息
