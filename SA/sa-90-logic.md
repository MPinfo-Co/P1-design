# 業務邏輯分析：公司背景功能

## 需求說明

讓使用者建立和維護「公司資料」，資安專家分析事件時需要參考的公司資訊集中管理，AI 夥伴分析事件時可參考使用者綁定的資料類別。

**核心使用情境：**

- 使用者希望 AI 夥伴能依公司實際環境做判斷
- 公司會持續累積背景知識與例外規則，需要簡單的編輯介面
- 同一類別的規則可同時被多個 AI 夥伴使用（雙向綁定）

---

## 商業邏輯

1. **逐筆管理規則**
   - 公司資料以列表式呈現，使用者可一筆一筆新增、修改、刪除、搜尋
   - 不把整個類別的規則塞進單一輸入框

2. **類別預設 3 種，可自由新增**
   - 預設 3 類：**環境 / 白名單 / 規則**
     - 環境：公司長什麼樣（網段、DMZ、設備清冊）
     - 白名單：不要管什麼（已知正常 IP/帳號/行為）
     - 規則：怎麼判斷處理（log 解讀、判斷門檻、處置 SOP）
   - 建立資料時可選既有類別，也可輸入新類別

3. **「適用夥伴」整類共享，改一筆會連動同類**
   - 同一類別所有規則共用一組「適用夥伴」
   - 例：把「環境」綁給資安專家後，環境類所有規則都能被資安專家讀取
   - 使用者改任一筆規則的「適用夥伴」時，同類別其他筆顯示的夥伴會一起變
   - 預設：資安專家綁所有 3 類

4. **兩個頁面都能改綁定，互相同步**
   - 「公司資料」頁與「資安專家設定」頁都能編輯「夥伴 ↔ 類別」
   - 在任一頁修改，另一頁立即反映

5. **預設類別有「內容」書寫範例**
   - 使用者選到預設 3 類時，新增 / 編輯對話框的「內容」欄位下方顯示範例提示：
     - 環境：例：DMZ 網段 10.0.1.0/24；用途：對外服務區
     - 白名單：例：svc_backup 每日 02:00 大量 IO 是正常備份
     - 規則：例：5 分鐘內登入失敗 ≥ 10 次，來源若為內網先忽略
   - 自訂類別不顯示（使用者自由發揮）

## 資料模型示意

| 實體                      | 欄位                 | 說明                     |
| ----------------------- | ------------------ | ---------------------- |
| `company_data`          | id, type, content  | 一筆 = 一條規則。             |
| `partner_data_bindings` | partner_name, type | 多對多關係：哪個夥伴可使用哪個類別的所有資料 |

**衍生顯示（不存成欄位，畫面顯示時即時計算）：**
- **類別下拉選單**：預設 3 類 + 使用者已用過的自訂類別（去重合併），即使預設類別目前沒有任何資料也仍要顯示
- **列表「適用夥伴」欄**：依該筆所屬類別，到 `partner_data_bindings` 反查出所有對應夥伴名稱

## SD 注意事項

1. **類別是自由字串，不是關聯表**
   - 使用者要能新增新類別
   - 預設 3 類由前後端共用同一份常數來源

2. **綁定以「類別」為單位**
   - 修改任一筆規則的「適用夥伴」= 覆寫該類別整組綁定
   - 同類別其他筆的「適用夥伴」顯示會連動更新

3. **雙向綁定靠共用資料來源**
   - 兩個 UI 入口讀寫同一份綁定資料，自然同步

4. **不做的功能（避免過度設計）**
   - 類別不做樹狀結構（單層即可）
   - 不做版本歷史 / audit log
   - 不做附件上傳（純文字）
   - 不做多語系
   - 「適用夥伴」勾選時不需要彈確認對話框

5. **與 Epic #131 的關係**
   - Epic #131「公司背景驅動的資安專家功能」會實際使用這份綁定資料做事件分析
   - 本 Epic 只負責建立資料 + 維護綁定，不負責「分析時怎麼用」

## 畫面/操作邏輯示意

雛型參考：`P1-design/Prototype/fn_company_data.html`（UI 細節以雛型為準）

### 畫面一：公司資料列表

- 列表式呈現（類似帳號管理頁），4 欄：類別 / 內容 / 適用夥伴 / 執行動作
- 「適用夥伴」欄顯示該筆所屬類別目前綁定的所有夥伴
- 支援關鍵字搜尋（類別 + 內容）

### 畫面二：新增 / 編輯資料對話框

- 欄位：類別（必填，可選既有或輸入新類別）、內容（必填）、適用夥伴（勾選列表）
- 選到預設 3 類時，「內容」欄位下方顯示對應範例提示（自訂類別不顯示）
- 副字提示使用者：「適用夥伴依類別共享」
- 儲存時：寫入該筆資料 + 覆寫該類別的整組夥伴綁定

### 畫面三：刪除確認

- 顯示確認對話框後刪除該筆
- 同類別綁定不因刪除而消失（即使該類別變成無資料）

### 與「資安專家設定」頁的雙向綁定

- 兩邊共用同一份綁定資料，任一頁修改後另一頁切換進入時反映最新狀態
