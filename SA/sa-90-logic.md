# 業務邏輯分析：[Epic] 公司背景功能

## 需求說明
## 功能說明

讓使用者建立和維護「公司背景知識庫」，把 AI 夥伴需要知道的公司資訊
（網段、伺服器、白名單、誤報規則等）集中管理起來。

每個知識庫包含：
- 內容：用白話寫的規則或背景知識（純文字）
- 設定誰可以編輯這個知識庫

預期會建立 3 個知識庫：
- 公司架構（網段、伺服器、DMZ 分區）
- 公司白名單（哪些是誤報、哪些服務帳號是正常的）
- 公司背景手冊（防火牆 log 對照、誤報案例等其他補充資訊）

## 驗收條件

- [ ] 知識庫列表頁可顯示、搜尋、新增
- [ ] 知識庫詳情頁可編輯：
  - 內容：用白話寫的規則或背景，可儲存
  - 存取設定：勾選哪些角色可以編輯
- [ ] 詳情頁顯示「綁定的 AI 夥伴」
- [ ] 後端資料表 + API 完整支援以上操作
- 雛型參考：P1-design/Prototype/kb.html
  （雛型有「文件」「資料表」兩個額外 tab，本 Epic 只做「內容」與「存取設定」）
