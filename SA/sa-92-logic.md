# 業務邏輯分析：資安專家設定功能

## 需求說明

「資安專家」是 MP-Box 內建的 AI 夥伴，負責看資安事件、給判斷與建議。
本次SA issue範圍：
1. 「AI夥伴管理」列表頁：以表格呈現所有 AI 夥伴（目前內建唯一夥伴：資安專家）
2. 「資安專家設定頁」：讓使用者控制資安專家怎麼運作：
   - 多久分析一次、什麼時間執行
   - 要連到哪台 SSB（log 收集主機）抓資料

---

## 商業邏輯

1. **AI 夥伴管理列表 → 點選進入該夥伴設定**
   - 從側邊欄「設定 > AI夥伴管理」進入夥伴列表頁
   - 列表以表格呈現，欄位：夥伴名稱 / 描述 / 啟用狀態 / 執行動作
   - 點該夥伴的「設定」按鈕，進入該夥伴的設定頁

2. **排程：什麼時候要跑分析**
   - 觸發時間：固定的某個時間點（時 / 分）
   - 觸發頻率擇一：
     - **每日**：每天到觸發時間自動執行
     - **每週**：選擇星期幾 + 觸發時間，每週該日到時間自動執行
     - **手動**：不自動跑，由「立即分析」操作觸發（立即分析功能本 Epic 不涵蓋）
   - 排程實際執行的流程不在此次issue範圍，本 Epic 只負責設定的儲存與讀取

3. **資料來源（syslog-ng Store Box）：要連到哪台主機抓 log**
   - 使用者填寫的連線欄位：Host、Port（預設 443）、Logspace、Username、Password
   - 系統固定（畫面顯示為資訊文字，不開放編輯）：
     - 通訊協定：HTTPS
     - API 版本：`/api/5/`
   - 「測試連線」按鈕可在尚未儲存的情況下，使用當前畫面填寫的值即時測試是否能連到 SSB
     - 成功：顯示連線成功訊息
     - 失敗：顯示錯誤訊息（含原因）

4. **儲存與還原**
   - 「儲存」會把畫面上的設定持久化
   - 下次進入設定頁時，欄位自動帶入上次儲存的值
   - 未儲存的修改在離開頁面後不保留

---

## SD 注意事項

1. **SSB 密碼安全性**
   - 密碼不可明碼存放，加密方式由 SD 設計
   - 回傳給前端的設定資料不應包含密碼明碼（以遮罩或不回傳呈現）

2. **「測試連線」不依賴儲存**
   - 測試連線 API 接受畫面當前輸入的 SSB 連線資訊作為參數，直接打 SSB 確認連線
   - 即使尚未儲存也能測；測試結果不影響資料庫狀態

3. **排程僅儲存設定，不執行排程**
   - 本 Epic 只做設定的介面，不實作排程器（cron / scheduler）
   - 「每週」觸發頻率需儲存星期幾欄位（0–6 或字串皆可，由 SD 決定）

4. **與其他 Epic 的關係**
   - 知識庫綁定 / 分析範圍：屬 Epic #131
   - 「立即分析」功能（按鈕 + 任意時間區間 from/to）：放在資安專家頁（非設定頁），由 Epic #131 處理
   - 通知方式：將另開獨立 Epic，本 Epic 不涵蓋
