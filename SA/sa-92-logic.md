# 業務邏輯分析：[Epic] 資安專家設定功能

## 需求說明
## 功能說明

「資安專家」是 MP-Box 內建的 AI 夥伴，負責看資安事件、給判斷與建議。
本 Epic 提供一個「資安專家設定頁」，讓使用者控制資安專家怎麼運作：

- 分析時要參考哪些公司背景知識庫
- 要分析的範圍（只看 Windows、只看 FortiGate、或全部都看）
- 多久分析一次、什麼時間執行
- 要連到哪台 SSB（log 收集主機）抓資料
- 有事件時用什麼方式通知（Email / Webhook / 系統內訊息）

## 驗收條件

- [ ] 進入設定頁直接看到資安專家設定（不用先選誰、也沒有列表）
- [ ] 4 區設定都可以編輯與儲存：
  - 區 A 分析行為：勾選要參考的知識庫、選分析範圍
  - 區 B 排程：設定觸發時間和頻率
  - 區 C 資料來源：填 SSB 連線資訊，可按「測試連線」確認
  - 區 D 通知：勾選通知方式，勾 Webhook 才會出現 URL 欄位
- [ ] 「儲存」後設定真的存進資料庫，下次進來還在
- [ ] 「測試連線」會實際打 SSB 確認能不能連
- 雛型參考：P1-design/Prototype/settings-ai.html

## 畫面/操作邏輯示意（選填）

**⚠️ ── AI 填寫開始，請逐行審查 ──**

### 資安專家設定頁

進入頁面後直接顯示設定表單，無列表、無夥伴選擇步驟。四個區塊由上至下依序排列，每區各自有標題區隔。頁面底部放置「儲存」按鈕。

---

**區 A — 分析行為**

- 知識庫綁定：列出系統現有知識庫，每筆顯示名稱，可多選勾選；若尚未建立任何知識庫，顯示提示文字並附連結引導前往建立。
- 分析範圍：以單選方式選擇要涵蓋的日誌來源，選項為「僅 Windows」、「僅 FortiGate」、「全部來源」。

**區 B — 排程**

- 觸發頻率：選擇分析執行間隔（如每日、每週）。
- 執行時間：選擇每次執行的起始時間（時 / 分）。

**區 C — 資料來源**

- 填寫 SSB 連線資訊：主機位址、連接埠、帳號、密碼。
- 「測試連線」按鈕：點擊後向 SSB 發起確認，成功時於按鈕旁顯示「連線成功」；失敗時顯示「連線失敗：{原因}」，原因由系統回傳。

**區 D — 通知**

- 通知方式以多選勾選呈現，選項為：Email、Webhook、系統內訊息。
- 勾選「Email」時，顯示 Email 收件人欄位（可填多組，以逗號分隔）。
- 勾選「Webhook」時，顯示 Webhook URL 輸入欄位；取消勾選後欄位隱藏。
- 未勾選任何方式時，不顯示額外欄位。

**儲存行為**

- 點擊「儲存」後，若有必填欄位未填，以提示文字標示於對應欄位下方，不執行儲存。
- 所有欄位驗證通過後，儲存成功顯示成功提示；儲存失敗顯示錯誤訊息。
- 下次進入頁面時，各欄位自動帶入上次儲存的值。
